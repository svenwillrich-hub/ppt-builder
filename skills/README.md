# Skills Directory

Place skill files here that Claude Code can reference during presentation generation.

## How to use

1. Drop `.md` or `.txt` skill files into this folder
2. The backend will automatically read them and include their content in prompts
3. Reference the skill name in your style configuration

## Example skills

- `capco-slides.md` — Capco-style presentation guidelines
- `consulting-pptx.md` — Consulting-style framework and visualization rules
