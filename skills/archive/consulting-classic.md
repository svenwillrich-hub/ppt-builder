# Consulting Classic Style

## Design Principles
- Clean, professional layout with generous whitespace
- Action titles: Every slide title is a key takeaway statement, not a topic label
- Pyramid Principle: Lead with conclusion, support with evidence
- Maximum 5 bullet points per slide, each under 15 words

## Visual Framework
- Use frameworks: 2x2 matrices, process flows, waterfall charts, Harvey balls
- Data-heavy: Include stats, percentages, benchmarks on every slide
- Source citations at the bottom of every slide with data claims
- Narrative arc: Situation > Complication > Resolution

## Typography
- Slide titles: 24-28pt, bold
- Body text: 14-16pt
- Source citations: 8-10pt, gray
- Use sans-serif fonts (Calibri, Arial, or similar)

## Layout Patterns
- Title slide: Bold statement + subtitle + date
- Content slide: Action title + 2-3 content blocks
- Data slide: Action title + chart/table + key insight callout
- Summary slide: 3-5 key takeaways with supporting evidence
