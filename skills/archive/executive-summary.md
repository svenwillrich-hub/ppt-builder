# Executive Summary Style

## Purpose
- Maximum 10 slides for C-level audience
- Each slide delivers exactly one key message
- Decision-oriented: every slide drives toward a recommendation

## Structure
1. Executive Summary (1 slide): 3-5 bullet takeaways
2. Situation (1-2 slides): Current state with data
3. Challenge (1-2 slides): Root causes and impact
4. Recommendation (2-3 slides): Proposed approach with rationale
5. Roadmap (1 slide): Timeline with milestones
6. Next Steps (1 slide): Immediate actions with owners

## Design Rules
- Large font sizes (20pt+ body, 28pt+ titles)
- Minimal text — use visuals to convey complexity
- Bold callout boxes for key numbers
- Use the "newspaper test": can you understand each slide in 10 seconds?
