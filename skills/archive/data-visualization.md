# Data Visualization Best Practices

## Chart Selection
- Comparison: Horizontal bar charts, grouped bars
- Trends over time: Line charts, area charts
- Part-to-whole: Stacked bars, waterfall charts, treemaps
- Relationships: Scatter plots, bubble charts
- Distribution: Histograms, box plots

## Design Rules
- Always label axes and include units
- Use the selected color palette for all chart elements
- Highlight the key insight with a callout or annotation
- Remove chart junk: no 3D effects, no unnecessary gridlines
- Sort bar charts by value (largest to smallest)

## Data Storytelling
- Every chart needs a "so what?" annotation
- Use before/after comparisons to show impact
- Highlight outliers and anomalies with contrasting colors
- Include source and date for all data points

## Common Frameworks
- 2x2 Matrix: Two dimensions, four quadrants with labels
- Waterfall: Show sequential gains/losses
- Harvey Balls: Rate maturity or completion levels
- Gap Analysis: Current vs. target with delta callout
